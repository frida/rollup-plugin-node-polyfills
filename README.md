@frida/rollup-plugin-node-polyfills
===

```
npm install --save-dev @frida/rollup-plugin-node-polyfills
```

Allows the node builtins to be `require`d/`import`ed.

Based on [rollup-plugin-node-polyfills](https://github.com/ionic-team/rollup-plugin-node-polyfills),
but ported to use Frida's existing shims.
